# Zen Chess – UI/UX Design

This repository contains the Figma design for the Zen Chess web application.

## 💻 Project Code
https://github.com/Kalpit7786/zen_chess

## 🎨 Figma Design Link

https://www.figma.com/design/vLnfkeVJhlPKkOtHsV5BvA/WebTech-ZENCHESS-figma?node-id=0-1&t=Su9SuBEHoZ9VbH8s-1

## 📌 Screens Included

* Login Page
* Register Page
* Dashboard
* Practice Mode (Chess Game)
* Vision Drill

## 🧠 Description

The design represents a frontend-based chess platform with AI gameplay and cognitive training features.
